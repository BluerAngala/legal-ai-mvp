【法律领域】
房产纠纷 - 房屋买卖、产权

【法律依据】
1. 《民法典》第二百零九条：不动产物权的设立、变更、转让和消灭，经依法登记，发生效力；未经登记，不发生效力，但是法律另有规定的除外。

2. 《民法典》第四百零六条：抵押期间，抵押人可以转让抵押财产。当事人另有约定的，按照其约定。抵押财产转让的，抵押权不受影响。抵押人转让抵押财产的，应当及时通知抵押权人。

3. 《民法典》第五百九十五条：买卖合同是出卖人转移标的物的所有权于买受人，买受人支付价款的合同。

4. 《城市房地产管理法》第三十八条：下列房地产，不得转让：
   - 以出让方式取得土地使用权的，不符合本法第三十九条规定的条件的
   - 司法机关和行政机关依法裁定、决定查封或者以其他形式限制房地产权利的
   - 依法收回土地使用权的
   - 共有房地产，未经其他共有人书面同意的
   - 权属有争议的
   - 未依法登记领取权属证书的
   - 法律、行政法规规定禁止转让的其他情形

【房屋买卖流程】
1. 签订买卖合同
2. 支付定金/首付
3. 办理网签
4. 银行按揭（如贷款）
5. 缴纳税费
6. 办理过户
7. 交房

【常见房产纠纷】
- 一房二卖
- 卖家违约涨价
- 买家逾期付款
- 中介违规操作
- 房产证办不下来
- 房屋质量纠纷
- 物业纠纷
- 户口未迁出
- 抵押房交易
- 继承房交易

【分析建议】
一、**买房前注意事项】
1. **查验房屋状况**
   - 产权清晰（无抵押、无查封）
   - 房产证、土地证
   - 实地查看房屋
   - 周边配套、规划

2. **查验卖方身份**
   - 产权人本人
   - 共有人同意
   - 代理人授权

3. **合同条款**
   - 房屋状况、价款
   - 付款方式、期限
   - 交房时间、条件
   - 违约责任
   - 户口迁出约定

4. **资金监管**
   - 第三方资金监管
   - 银行托管
   - 不要直接给卖家大额现金

二、**卖方违约不卖了**
1. 继续履行（强制过户）
2. 解除合同+违约金
3. 双倍返还定金
4. 赔偿损失

三、**房屋质量问题】
- 主体结构质量不合格：解除合同+赔偿
- 一般质量问题：修理+赔偿
- 隐瞒质量：欺诈，可撤销合同

四、**无法办理房产证**
- 开发商原因：要求继续履行+赔偿
- 政策原因：协商解决
- 历史遗留：寻求政府解决

五、**物业纠纷】
- 物业服务质量：业主委员会协商
- 物业费：按合同约定
- 维修基金：按规定使用

【诉讼时效】
- 房产纠纷：3年
- 逾期交房/办证：约定期限届满之日起算
